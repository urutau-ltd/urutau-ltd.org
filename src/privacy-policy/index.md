# ACUERDO DE PRIVACIDAD Y COMPROMISO DE CERO RECOLECCIÓN DE DATOS PERSONALES

## 1. Preámbulo y Principio Fundamental

En Urutaú Limited, la privacidad y el respeto absoluto por la identidad digital
de nuestros visitantes son nuestros principios rectores. Reconocemos la
importancia de la privacidad en línea y estamos comprometidos a operar nuestro
sitio web de una manera que minimice cualquier potencial de recolección, uso o
almacenamiento de información personal.

Este sitio web es de naturaleza puramente estática, está diseñado
intencionalmente para no recopilar, rastrear, registrar o almacenar datos
personales de los usuarios. Nos comprometemos a mantener un entorno
completamente libre de seguimiento (_tracking-free_) para todos los visitantes.

## 2. Alcance y Aplicación del Presente Acuerdo

El presente Acuerdo de Privacidad se aplica exclusivamente a la información
generada o recopilada (o, en nuestro caso, la NO recopilación) a través del
acceso y uso de Urutaú Limited. Este compromiso cubre todas las interacciones
que el usuario tiene directamente con el código fuente.

## 3. Cero Recolección de Datos Personales del Lado del Cliente

Urutaú Limited **NO** recopila, registra, almacena ni procesa ninguna de las
siguientes categorías de información personal, sensible o de rastreo del lado
del cliente:

- **Identificadores Directos**: No solicitamos ni almacenamos nombres,
  direcciones de correo electrónico, números de teléfono, direcciones postales o
  cualquier otra información de contacto personal.

- **Datos de Autenticación/Cuentas**: No existen cuentas de usuario, registros
  ni credenciales de acceso.

- **Datos de Navegación/Comportamiento (Analytics)**:
  - No utilizamos Google Analytics, Matomo, Cloudflare Analytics ni ningún otro
    sistema de análisis de terceros o propio.
  - No utilizamos cookies (propias o de terceros) con fines de rastreo,
    analíticos, publicitarios o de personalización.
  - No empleamos tecnologías de huella digital del navegador (fingerprinting),
    pixels de seguimiento, web beacons o localStorage para almacenar
    identificadores de seguimiento.
  - No rastreamos ni registramos el historial de páginas visitadas, el tiempo de
    permanencia, los clics, el desplazamiento (scrolling) o cualquier otra
    interacción del usuario dentro del sitio.

- **Datos Técnicos del Dispositivo (Direcciones IP y Logs del Lado del
  Servidor - Ver Sección 4)**:
  - La dirección IP y los datos de registro de servidor (logs) no son
    accesibles, consultados, procesados o retenidos por el Responsable del sitio
    web para identificar a los usuarios o para fines de seguimiento. (Consulte
    la Sección 4 sobre la gestión de Logs del Servidor).

- **Datos de Ubicación**: No se solicita, rastrea ni almacena ninguna
  información de geolocalización.

- **Datos Personales Sensibles**: No se recopila intencionalmente ninguna
  categoría de datos sensibles (salud, origen racial, orientación sexual,
  creencias, etc.).

- **Formularios o Interacciones con el Usuario**: No hay formularios de
  contacto, de suscripción, de comentarios o de otra índole diseñados para
  recopilar datos del usuario.

## 4. Gestión de Logs del Servidor (Datos Técnicos y Operacionales)

- **Definición**: Los logs del servidor son archivos automáticos generados por
  el proveedor de hosting (alojamiento) que registran la actividad de la red
  (como la hora de acceso, el navegador utilizado, el referrer y la dirección
  IP) por motivos técnicos esenciales de seguridad, estabilidad y diagnóstico de
  errores.

- **Compromiso del Responsable**: Aunque la generación de estos logs es una
  función estándar del hosting sobre la cual el Responsable tiene control
  limitado o nulo (es inherente a la infraestructura de internet), el
  Responsable se compromete a:

- **No Acceder ni Procesar**: El Responsable no accede, utiliza, ni procesa
  estos logs de servidor para el análisis del tráfico, rastreo de usuarios o
  cualquier finalidad que contravenga el compromiso de "cero recolección".

- **Retención Mínima**: Si el proveedor de hosting retiene estos logs, la
  retención se considera una necesidad técnica de dicho proveedor. El
  Responsable aboga por la retención mínima posible (idealmente 7 días o menos)
  y la anonimización o pseudonimización estricta por parte del proveedor.

- **Seguridad**: En caso de que estos logs contengan direcciones IP, se tratarán
  como información altamente sensible bajo las medidas de seguridad más
  estrictas para evitar el acceso no autorizado.

## 5. Transferencia de Datos y Terceros Proveedores de Servicios

Dado que no recopilamos datos personales, NO existen datos personales que puedan
ser transferidos, compartidos, vendidos o alquilados a terceros.

- **Proveedor de Alojamiento (Hosting)**: La única entidad que interactúa
  indirectamente con la información técnica necesaria para la visualización del
  sitio (como la dirección IP en los logs del servidor) es nuestro proveedor de
  hosting: Namecheap, Inc. Sin embargo, bajo este Acuerdo, el sitio web no
  depende de ningún servicio de terceros para el rastreo.

- **Contenido Incrustado o Servicios Externos**: Este sitio web se abstiene de
  utilizar contenido incrustado de terceros (como videos de YouTube o Vimeo,
  feeds de redes sociales, widgets de comentarios) que pudieran iniciar la
  recolección de datos por parte de dichos terceros al cargar el contenido.

## 6. Bases Legales para el Tratamiento de Datos (NO APLICABLE)

De conformidad con el principio de mínima recolección, NO existe base legal
necesaria para el tratamiento de datos personales, ya que no se recogen ni se
tratan datos personales. Este sitio web opera sobre la base del no-tratamiento
de información que pueda identificar a un individuo.

## 7. Derechos del Usuario (Derechos ARCO, RGPD, LFPDPPP, etc.)

Dado nuestro compromiso de Cero Recolección de Datos Personales, la aplicación
práctica de los derechos de Acceso, Rectificación, Cancelación u Oposición
(Derechos ARCO, o derechos análogos bajo otras jurisdicciones como el RGPD) se
ve afectada, ya que no tenemos datos personales para acceder, rectificar o
suprimir.

- **Derecho a la Información**: Se cumple con el presente Acuerdo de Privacidad,
  que detalla la política de no-recolección.

- **Otros Derechos (Acceso, Rectificación, Supresión, Portabilidad)**: El
  ejercicio de estos derechos es irrelevante o no aplicable, pues no poseemos ni
  almacenamos la información personal del usuario. Si un usuario alega que sus
  datos personales han sido recogidos a través de este sitio, el Responsable
  investigará inmediatamente la infraestructura para corregir cualquier anomalía
  que contravenga este Acuerdo.

## 8. Cambios a este Acuerdo de Privacidad

Este Acuerdo de Privacidad podrá ser modificado en el futuro. Cualquier cambio
será publicado en esta misma página con una nueva "Fecha de Última
Actualización" para asegurar la máxima transparencia. Si se planea añadir
cualquier forma de recolección de datos (analytics, formularios, cookies de
seguimiento), se notificará con anticipación y se solicitará el consentimiento
explícito del usuario según lo exija la ley aplicable, invalidando de facto la
premisa de este documento.

## 9. Aceptación y Consentimiento

Al acceder y utilizar el sitio web de Urutaú Limited, usted reconoce haber leído
y entendido este Acuerdo de Privacidad y acepta que no se recopilará, rastreará,
almacenará ni procesará ninguna información personal identificable sobre usted
por parte del Responsable, a excepción de los logs técnicos e inevitables del
servidor generados por el proveedor de hosting (ver Sección 4), que no serán
utilizados para rastreo.
