# Security Policy

This is a static site, but it still ships client-side code, a service worker,
build tooling, and public contact metadata. If you find a vulnerability that
affects the generated website, the published assets, or this repository's
tooling, report it through the public contact path on the website:

- https://urutau-ltd.org/contact/

The canonical machine-readable contact entry is published at:

- https://urutau-ltd.org/.well-known/security.txt

Please avoid filing sensitive reports in public issue trackers.
